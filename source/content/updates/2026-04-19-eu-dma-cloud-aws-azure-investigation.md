## EC Launches DMA Gatekeeper Investigations Into AWS and Microsoft Azure Cloud Services

- **Date**: 2026-04-19
- **Region**: EU
- **Company relevance**: Sector-wide — Stack Overflow, Swiggy, PayU, iFood, Codecademy, Eruditus, OLX and all Prosus companies on AWS or Azure
- **Priority**: HIGH
- **Source**: [European Commission / CIO Dive](https://www.ciodive.com/news/eu-regulators-launch-cloud-market-probes/805841/)

The European Commission has opened three separate investigations under the Digital Markets Act to assess whether Amazon Web Services and Microsoft Azure should be formally designated as gatekeepers in cloud computing. A first investigation assesses AWS's gatekeeper status; a second assesses Microsoft Azure; a third is a market-wide investigation into whether the DMA is effective at addressing anti-competitive practices in cloud — specifically targeting egress fees, software licensing lock-in, and proprietary API dependencies. Google Cloud was not included in this round. The Commission has a 12-month deadline to conclude the specific designation investigations. If designated, AWS and Azure would face DMA obligations including mandatory data portability, interoperability with competing cloud services, prohibition on technical or contractual switching barriers, and fair access to metering/monitoring data. Fines for non-compliance can reach 10% of global annual turnover. This directly follows the UK CMA's ongoing cloud market investigation (provisional findings expected Q3 2026), creating a dual-jurisdiction enforcement environment for the same platforms. For Prosus portfolio companies, designation would meaningfully reduce switching costs and open negotiating leverage in cloud contracts.
